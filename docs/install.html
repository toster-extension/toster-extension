Установка

Google Chrome

Установка из магазина расширений

  1. Перейти по ссылке toster-extension
  2. Нажать кнопку Установить
  3. В появившемся окне нажать кнопку Установить расширение

Установка в режиме разработчика

  1. Перейти по ссылке и скачать файл extension.zip
  2. Распаковать скачанный архив
  3. Ввести в адресной строке браузера chrome://extensions/
  4. Включить Режим разработчика
  5. Нажать кнопку Загрузить распакованное расширение
  6. Выбрать каталог, куда был распакован архив и нажать кнопку Открыть

Firefox

Установка из магазина расширений

  1. Перейти по ссылке toster-extension
  2. Нажать кнопку Добавить в Firefox
  3. В появившемся окне нажать кнопку Добавить

Установка из файла

  1. Перейти по ссылке и скачать файл extension.xpi
  2. Ввести в адресной строке браузера about:addons
  3. Нажать кнопку с шестеренкой Инструменты для всех дополнений
  4. Выбрать пункт Установить дополнение из файла
  5. Выбрать скачанный файл extension.xpi и нажать кнопку Открыть
  6. В появившемся окне нажать кнопку Добавить
